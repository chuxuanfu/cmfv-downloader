# 两个视频下载工具对比

你现在有两个视频下载工具，分别用于不同的场景：

## 📦 工具对比

| 特性 | apple-video-downloader | cmfv-downloader |
|------|------------------------|-----------------|
| **用途** | HLS 流媒体 (.m3u8) | CMAF 分段 (.cmfv/.cmfa) |
| **格式** | M3U8 + M4S 分段 | Init + CMFV/CMFA 分段 |
| **典型应用** | Apple Event HLS 流 | Apple CDN CMAF 视频 |
| **下载方式** | 顺序下载（可靠） | 并发下载（快速） |
| **断点续传** | ✅ 支持 | ✅ 支持 |
| **速度** | 慢但稳定 | 快但需要配置 |
| **复杂度** | 简单（自动） | 需要手动配置 URL |

## 🎯 使用场景

### apple-video-downloader

**适用于：**
- ✅ 标准 HLS 流媒体（.m3u8）
- ✅ 不想手动配置 URL
- ✅ 网络不稳定的环境
- ✅ 需要高可靠性

**示例：**
```bash
cd /Users/chuxuanfu/apple-video-downloader
./download_apple_video.sh
```

直接提供 M3U8 URL 即可，脚本自动处理一切。

### cmfv-downloader

**适用于：**
- ✅ CMAF 格式视频（.cmfv/.cmfa）
- ✅ 需要快速下载大量分段
- ✅ 网络稳定的环境
- ✅ 能够手动获取 URL 模板

**示例：**
```bash
cd /Users/chuxuanfu/cmfv-downloader
# 编辑 download_merge.sh 配置 URL
./download_merge.sh
```

需要手动配置 URL 模板，但下载速度更快。

## 📋 技术细节

### HLS (apple-video-downloader)

```
结构:
master.m3u8 → 指向不同画质的 playlist
playlist.m3u8 → 包含所有 .m4s 分段的列表

下载方式:
1. 解析 m3u8 获取分段列表
2. 顺序下载每个分段（带重试）
3. 拼接分段
4. 混流
```

### CMAF (cmfv-downloader)

```
结构:
init.cmfv → 初始化信息
seg_000000001.cmfv → 第1个视频分段
seg_000000002.cmfv → 第2个视频分段
...

下载方式:
1. 手动配置 URL 模板
2. 并发下载所有分段
3. 二进制拼接
4. 混流
```

## 💡 选择建议

### 优先使用 apple-video-downloader

**如果你：**
- 🔰 第一次下载
- 🌐 网络不太稳定
- 🎯 想要简单快速
- 📺 下载标准 HLS 流

### 使用 cmfv-downloader

**如果你：**
- 🔧 熟悉浏览器开发者工具
- 🚀 需要更快的下载速度
- 💻 有稳定的网络连接
- 📹 视频是 CMAF 格式
